# Tarefa

Criar uma tabela com o tempo de execução de 8 algoritmos de ordenação (inspire-se na tabela do material de aula). 4 são as implementações básicas de cada um dos algoritmos (bubblesort, selectionsort, insertionsort e shellsort), os outros devem conter otimizações de cada método ou formas recursivas.

Junto ao tempo, entre parêntese, colocar a diferença de tempo relativa ao Bubble Sort base de cada entrada.

Criei os 4 arquivos de bubble, insertion, selection e shellsort em suas versões normais e otimizações.

A tabela pode ser vista ao rodar o arquivo tabela_de_tempo.py, que chama as outras funções, mostra o tempo e a porcentagem de diferença em relação ao Bubble Sort base.
